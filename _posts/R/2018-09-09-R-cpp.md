R에서 C++ 시작하기
------------------

C++ 함수의 간단한 활용법에 대해서 알아보자. Rcpp를 설치한 후에 CppFunction안에 간단한 C++ source를 작성한다.

``` r
if (!require(Rcpp)) install.packages("Rcpp")
cppFunction("
            int add(int x, int y, int z) {
              int sum = x + y + z;
              return sum;
            }
            ")
add
```
