Nvim-R의 shortcut
----------------
 - ubuntu bionic-beaver 18.04 기준

Nvim-R 설치 후 R 파일을 open 해보자.

``` console
sudo vim test.R
```

R 파일에 간단한 내용을 작성해보자.

``` r
library(ggplot2)
head(cars)
ggplot(data = cars, aes(x = speed, y = dist)) + geom_line()
```
\를 누른 상태에서 rf를 누르면 R 프롬프트 창이 켜진다.  
그 상태에서 \pp를 누르면 파일이 실행된다.
